# 🎥 Movie Watchlist App

A sleek and modern movie search application that allows users to discover films and build their personal watchlist in real-time. Designed with a dark-themed UI inspired by popular streaming platforms.

---

## 🚀 Features

* 🔍 **Live Movie Search**
  Instantly fetch and display movie results as the user types or clicks search.

* 📌 **Watchlist Toggle**
  Add or remove movies from your personal watchlist with a single click.

* 📭 **Empty States Handling**

  * Displays a message when no movies are found
  * Shows a clean UI when the watchlist is empty

* 📺 **Dynamic Movie Cards**
  Each movie includes:

  * Poster
  * Title
  * Year
  * Type (Movie/Series)

* 📌 **Sticky Watchlist Sidebar**
  A fixed sidebar that updates in real-time as users add/remove movies.

## 🌐 APIs Used

* 🎬 **OMDb API** (Primary search data)
  👉 [https://www.omdbapi.com/](https://www.omdbapi.com/)

* 🎞️ **TMDB API** (Optional enhancements like posters/details)
  👉 [https://developer.themoviedb.org/docs/getting-started](https://developer.themoviedb.org/docs/getting-started)

* 📡 **Watchmode API** (Optional streaming availability)
  👉 [https://api.watchmode.com/](https://api.watchmode.com/)

---

## ⚙️ How It Works

1. User types in the search bar
2. App fetches movie data from OMDb API
3. Results are displayed instantly as cards
4. User can:

   * Add movies to watchlist
   * Remove movies from watchlist
5. Watchlist updates live in the sidebar
